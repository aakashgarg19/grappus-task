# GrappusTask

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.1.2.

## Project Setup
1. Install Node.js (`https://nodejs.org/en/download/`).
2. Go to project Repository From Command Line.
3. Run command `npm install`.
4. After Successful Installation, Run Command `ng serve`.
5. Navigate to `http://localhost:4200/` in the browser.
